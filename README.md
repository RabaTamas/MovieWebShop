# MovieWebShop

I developed a webshop selling movies on ASP.NET Core and React platforms. I used these technologies:
 - Backend: ASP.NET Core:
    - Entity Framework Core
    - ASP.NET Identity
    - AutoMapper
    - Authorization Policies
    - Dependency Injection
    - Service Layer
    - Swagger UI
- Frontend:
    - React
    - HTML, CSS
- SQL Server
- JSON Web Token
- REST Api
- JSON
- TMDB Api
